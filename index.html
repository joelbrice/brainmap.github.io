BrainMap API Documentation

Analyze MRI images with our prediction API

Overview

BrainMap is an API designed for analyzing MRI images. It provides a single endpoint for making predictions based on uploaded MRI data.

API Endpoint

Predict

POST /predict

Uploads an MRI image and returns a prediction using the latest trained model.

Request

  • Method: POST
  • Content-Type: multipart/form-data
  • Body: Form data with a file field named 'file' containing the MRI image

Response

JSON object containing:

  • Prediction: The predicted value
  • Probability: The probability of the prediction

Usage Example


curl -X POST -F "file=@path/to/your/image.jpg" https://api.brainmap.com/predict

Replace 'path/to/your/image.jpg' with the actual path to your MRI image file.
