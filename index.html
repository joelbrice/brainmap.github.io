BrainMap API Documentation

Overview

BrainMap is an API for processing and analyzing MRI data. It provides endpoints for preprocessing, training models, evaluating performance, and making predictions.

Endpoints

1. Preprocess

POST /preprocess

Preprocesses raw MRI data and stores the processed data.

2. Train

POST /train

Trains a model on preprocessed data and returns the validation MAE.

3. Evaluate

POST /evaluate

Evaluates the performance of the latest production model on processed data.

4. Predict

POST /predict

Makes a prediction using the latest trained model.
